# My DSA Journey

This repository tracks my journey learning Data Structures and Algorithms, based on [ThePrimeagen's kata-machine](https://github.com/ThePrimeagen/kata-machine) and the [Frontend Masters Algorithms Course](https://theprimeagen.github.io/fem-algos/).

## Setup

Install dependencies:
```bash
yarn install
```

## Usage

Generate a new day of exercises:
```bash
yarn generate
```

Run tests:
```bash
yarn test
```
